One of the many things you can do with SECURElogBook is to create periodic reminders. Thanks to its built-in time calculation features, setting up these reminders is a breeze. You do not need to worry, as you can be sure that the time calculations will be done without any issues.

Let's say we need to take a total of 6 pills and we need to take the medicine every 8 hours.

We can create reminders one by one and waste time doing mental calculations or we can use the SECURElogBook functions that will make things easier for us. We can also check if what we have done is correct.
