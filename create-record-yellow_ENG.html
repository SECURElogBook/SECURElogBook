Event registration (yellow record)

It is also possible to create a log to track something that is still running, like:

 . How long can you exercise?
 . How much sleep do you need to wake up without an alarm?
 . How long does it take for your cake to cool?
 . How long does it take you to cook your new recipe?

When the task is completed, you can add the time to the record and that information will be yours for ever. Once you know that information, you can focus on the next thing that can be improved. That is the idea behind SECURElogBook.

This is also the type of record you can create if you have a great idea or just you realised you must do something today, but you do not have time to place it in the right time. You just want to remember it and make sure you check it out today as it will be highlighted as a yellow record. It is a good practice to review your yellow records one or two times a day.

1. Open SECURElogBook and go to the main window, with blue background:


2. Press the button (I), to add a new record/task.

3. Once the creator window (green background) is opened we were able to create our record/task:


The small list with all the records (K) will show only records that match or follow the moment selected by time and date (D and F).

4. By default, the record time is set to the present and the task is set as not completed. That is the reason why the record buttons (H, I, J) are yellow.

When creating a record it is useful to have the current time set as the time of our record by default. Note that the phone time is the same as our record, 21:19. Thanks to that we can easily add or subtract time when we use the button to open the windows to help us manage time:

These windows help you to add or subtract time without the possibility of making mistakes when you press this button

5. Add the text inside the text box. If you save a record/task with no text, the program will add “NO TEXT” by default.

6. Because we are recording an event that is still running, we don't need to do anything. A new record, by default, is set to the current time and not completed. Therefore, the recording buttons (H, I, J) will be yellow.

The colour of the record buttons (H, I, J) is like a preview of how the record we are creating will look like.

7. Now that we have a yellow record we can save it, by pressing the button (H).

8. If the records buttons turn red, is because there is already a record set with that date and time. That means you cannot save your record there. This record with the same time, will be the first one in the list or records. If that happens, simply select another time to save the record.

9. Save the record by pressing the button H. You will see a confirmation message with a black background:

 . Black = past.
 . Red = today.
 . Blue = tomorrow.
 . Green = beyond the day after tomorrow.

EXAMPLE OF POSSIBLE ERROR: if you created a record to register an event in the past but the confirmation message was not black you have a problem. Because you created a record in the future, and it will have an alarm. You can always press the button on the main window to open the last saved record to fix the problem.

Congratulations you just saved your first event.
