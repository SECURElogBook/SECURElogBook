Shopping list/lists

SECURElogBook also allows to create lists. Once a list is created you will be able to move and copy it.

1. Open SECURElogBook and go to the main window, with blue background:


2. Press the button (I), to add a new record/task.

3. Once the creator window (green background) is opened we were able to create our record/task:


The small list with all the records (K) will show only records that match or follow the moment selected by time and date (D and F).

4. By default, the record time is set to the present and the task is set as not completed. That is the reason why the record buttons (H, I, J) are yellow.

When creating a record it is useful to have the current time set as the time of our record by default. Note that the phone time is the same as our record, 21:19 (image above). Thanks to that we can easily add or subtract time when we use the button to open the windows to help us manage time:

These windows help you to add or subtract time without the possibility of making mistakes.

5. You can add any text but let’s call this list “Shopping list”:


6. Now that we have a yellow record we can save it, by pressing the button indicated with the yellow arrow.

7. Tap on the “Shopping list” record to edit it:


8. The editor window (teal background) will display the “Shopping list” record.


By default, when we edit a record, the record will be the first one on the editor window list of records.

9. Type or select from favourites the item you want to add. In this case we are going to add “Bread”:


10. Press the button with the "L" (list) to add the item to the list:


Once the item is added the text box will be cleared.

11. Now select from the menu or type "Eggs", then press the button with the "L" (list) to add the item to the list:


12. Now select from the menu or type "Milk", then press the button with the "L" (list) to add the item to the list:


Note that the items in the list will be sorted alphabetically.

13. Press the back button on your phone to go back to the main window:


14. Tap on the "Shopping list" record to edit it:


Once we have a list, items are attached to the first record, "Shopping list". If we can use that record to move in time or create a copy of the entire list.

15. To complete an item from the list we can HOLD on it. HOLD your finger on "Eggs":


We can also complete or not complete every record in the list. For example we can indicate that "Eggs" are already in the trolley.
12. Press the back button on your phone to complete the list and go to the main window.

Once the list is created you can use the main record “Shopping list” to copy or move the entire list.

You can set items inside the list as completed/not completed independently.
